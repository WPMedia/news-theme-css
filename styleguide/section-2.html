2 Grid

The Grid framework is a bootstrap-like row/column convention, but is implemented in CSS Grid.

In the examples give below, borders are given to containers (green), rows (red) and column content (blue)

Source: grid/_grid_containers.scss, line 1

2.1 .container-fluid

Provides a full-width container. Useful for Header and Footer

Example

This is a container fluid 
  <header class="container-fluid">
      This is a container fluid
  </header>

Source: grid/_grid_containers.scss, line 18

2.2 .container

Provides a centered container with margins based on breakpoints. Useful for wrapping a row class (CSS Grid)

Margins stay at 5% for sm and md grid breakpoints, for lg they are 8% and auto for xl

Example

This is a container 
  <div class="container">
      This is a container
  </div>

2.4 .col-[breakpoint]-[col#]

Provides column classes that are immediate children of rows. For each breakpoint defined, and for each column number, there is a corresponding class to indicate how many columns to span for each breakpoint
